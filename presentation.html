Accounting
محاسبة احترافية تلقائية
تسحب البيانات تلقائياً من جميع وحدات النظام وتُنتج قوائم مالية معيارية فورياً دون أي إدخال يدوي
قائمة الأرباح والخسائر
  • إيرادات الفواتير المفعّلة بالفترة
  • تفصيل المصروفات حسب الفئة
  • الرواتب × عدد أشهر الفترة
  • صافي الربح أو الخسارة
  • فلتر بالسنة والشهر
الميزانية العمومية
  • أصول متداولة (نقد + ذمم مدينة)
  • أصول ثابتة (قيمة المعدات)
  • خصوم وحقوق الملكية
  • رأس المال + الأرباح المحتجزة
  • فحص توازن تلقائي
ميزان المراجعة
  • جميع الحسابات بالمدين والدائن
  • إجمالي الميزان مع فحص التعادل
  • توزيع المصروفات بالنسب المئوية
  • أشرطة تقدم بصرية للمقارنة
  • تصنيف كل حساب (أصل/إيراد/مصروف)
شجرة الحسابات
  • دليل حسابات قياسي (1000 → 5000)
  • أصول، خصوم، ملكية، إيرادات، مصروفات
  • أرصدة حية مستخرجة من قاعدة البيانات
  • شرح لكل مجموعة حسابات
  • مبني على المعايير المحاسبية الدولية

Tech Stack
المكدس التقني

PHP 8.1
الخادم الخلفي الرئيسي · معالجة الطلبات وقواعد البيانات والمصادقة. يعتمد على PHP الحديث مع دعم كامل للـ OOP والدوال المساعدة.

قاعدة البيانات الرئيسية · 14 جدول مترابط بعلاقات FK. ترميز UTF-8mb4 للعربية بشكل كامل وصحيح.

PWA — Progressive Web App
Service Worker + Web Manifest · يُثبَّت على الجوال. يعمل على iOS وAndroid كتطبيق أصلي بدون متجر تطبيقات.

GitHub Actions CI/CD
نشر تلقائي عند كل تحديث · FTP Deploy Action. دورة النشر الكاملة في أقل من 60 ثانية تلقائياً.

SSL/TLS + HTTPS
